# Marten

Http chaos monkey server.

## Requirements
    eve
    eve-swagger
    eve-sqlalchemy
    
## Setup
    virtualenv venv
    source venv/bin/activate
    pip install -r requirements.txt

## CONFIG
    
    export AUTO_RELOAD=True
    export DEBUG=True
    export HOST=True
    export PORT=True
    
## RUN

    ./run.py
